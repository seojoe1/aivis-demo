# AIVIS - Design Demo

A static, self-contained design demo of **AIVIS**, an AI search visibility and brand-perception
platform that tracks how a brand appears in AI answers (ChatGPT, Perplexity, Google AI Overviews)
rather than traditional search rankings.

**Live demo:** https://seojoe1.github.io/aivis-demo/

All figures are illustrative demo data for a fictional brand (Clarendon Homes). No real client data.
Every page is a self-contained HTML mockup (inline CSS/JS, embedded fonts, no external requests).
