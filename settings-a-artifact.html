Brand Settings mode · 12 sections

    
1 Profile - name, category, market, plan price (owner-only)

    
2 Domains - brand_domains, add/remove, canonical/owned/excluded, one primary

    
3 Competitors & aliases - per-alias match type + case/word-boundary flags

    
4 Exclusion filters - term/topic, manual vs auto source

    
5 Live matcher test - paste answer, live highlighted matches (no persistence)

    
6 Engines & sampling - engines, runs/prompt, cadence, pulse, budget cap

    
6b Prompt portfolio - versions list, active pin, edits create a new version

    
7 Facts manager - shortcut to Accuracy with live fact count

    
8 Imports - multi-step wizard, modes, upload, field mapping

    
9 Integrations - GA4 / Search Console connections

    
10 Crawler tracking - ingest key + install snippets, one-click copy

    
11 Danger zone - archive brand, type-to-confirm gate


    
Org Settings mode · 11 sections (5 legacy pages + audit log incorporated)

    
1 Profile - org name, contact email, read-only slug

    
2 Team - member list, invite, resend, deactivate Legacy

    
3 Billing - subscription status, checkout + Stripe portal Legacy

    
4 Branding - white-label logo/colour with live preview Legacy

    
5 Security & MFA - TOTP enrol, factors, recovery (user-scoped) Legacy

    
6 Webhooks - create/delete, event type, endpoint + secret Legacy

    
7 Notifications - Slack webhook, alert emails, test send

    
8 Data retention - raw response + report retention windows

    
9 Feature flags - per-flag toggle with org/brand/user scope

    
10 Audit log - paginated action history, read-only, admin-only Legacy

    
11 Export & danger zone - export, delete-client + delete-org type-to-confirm (owner)
